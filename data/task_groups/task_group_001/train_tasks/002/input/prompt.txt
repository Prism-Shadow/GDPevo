HarborCRM is the shared CRM marketing workspace for this task group. The local API is available after starting `env/setup.sh`; use the base URL supplied by the runner, or `http://127.0.0.1:8067` when running locally.

Prepare the qualified lead list for an OEM dissolved-oxygen sensor campaign using the MarineSense Expo 2026 trade-show records (`show_id`: `marinesense_2026`). Use the public trade-show, CRM, meeting-interest, and policy data exposed by HarborCRM. Return the result as JSON only, following `input/payloads/answer_template.json`.

The output should identify the import-ready qualified exhibitors, classify their applicable platform coverage with the allowed platform enums, preserve the requested enrichment fields, record important non-qualified near misses, and summarize the resulting counts.
