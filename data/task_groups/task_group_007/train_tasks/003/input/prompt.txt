Northwind Components needs a Q1 supplier incident scorecard for the weekly supplier-quality review.

Use the shared ERP API described in the task group environment. The API base URL will be provided by the runner, or you can start the local environment and use `http://127.0.0.1:8007`. Relevant public endpoints include `/incidents` and `/suppliers`.

Prepare the scorecard using the request details and recommendation policy in `input/payloads/q1_scorecard_request.json`. Return a JSON object that conforms to `input/payloads/answer_template.json`.

The scorecard must include the filtered incident population, supplier-level counts and percentages, total resolution cost, average duration, incident-type split, open/severe counts, and the controlled recommendation code for each supplier with Q1 incidents.
