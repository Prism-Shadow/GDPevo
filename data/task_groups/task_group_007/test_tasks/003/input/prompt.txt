Northwind Components needs a full-year supplier incident analysis for the annual supplier-quality and operations review.

Use the shared ERP API described in the task group environment. The API base URL is provided by the runner as `<TASK_ENV_BASE_URL>`. Relevant public endpoints include `/incidents` and `/suppliers`.
